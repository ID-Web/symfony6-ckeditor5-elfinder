# Symfony6, CKeditor5 and elFinder
Working example of CKeditor 5 (w/o bundle) + Elfinder  in a symfony 6 project

How to use:

Clone the repo
$ git clone 

install stuffs
Composer install
npm install

build the app:
./node_modules/.bin/encore dev
